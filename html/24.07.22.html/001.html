네트워크의 시작

Local Area Network(LAN)의 시작

  1. 저장장치 공유
  2. 프린터 공유

인터넷의 등장

  • LAN들을 연결
  • 한쪽 선에 문제가 생겨도 우회해서 통신이 가능
  • router : 데이터의 목적지가 우리면 흡수, 아니면 목적지 쪽으로 패스 (우리꺼면 받고 아니면 내침)

인터넷과 서비스

인터넷을 위한 통신규약 : TCP/IP - 이 위에서 SMTP,HTTP,FTP가 돌아간다

  • 이메일 서비스 : smtp 프로토콜
  • WWW(World Wide Web) : http 프로토콜
  • 파일 다운로드 : ftp 프로토콜

Internet, intranet, extranet

  • Intranet(인터넷 기술을 사용한 내부망) : TCP/IP를 사용하는 LAN
  • Extranet(인트라넷 끼리 연결 중 생길 정보유출 위험을 방지) : 인트라넷을 암호화 장비로 연결
